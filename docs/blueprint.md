# **App Name**: RoomRadar

## Core Features:

- Location Tracking: Use the device's GPS to pinpoint the tenant's current location.
- Map Integration: Integrate with Google Maps API to display nearby available rooms and buildings.
- Property Listings: Allow landlords to post building with room listings including details like price, number of rooms, amenities, and photos.
- Room Search: Enable tenants to search for rooms based on location, price, amenities, and other filters.
- Tenant-Landlord Communication: Enable communication platform for Tenant and Landlord
- Smart Listing Enhancement: Employ a tool to analyze listings, suggest improvements to descriptions and tags based on similar, successful postings in the area.
- My Map: Searching the nearby building and rooms listed on the map.
- My Booking: Showing the rooms that has been booked.
- My Listing: Listing the building and the room.
- My Wallet: Viewing the balance and transaction history as well as top up and withdraw.
- My Message: Exchanging messages between landlord user and tenant user.

## Style Guidelines:

- Primary color: Muted blue (#5DADE2) to convey trust and stability in the rental process.
- Background color: Very light blue (#EBF4FA), providing a calm backdrop.
- Accent color: Soft orange (#E59866) to highlight interactive elements.
- Body and headline font: 'PT Sans', a humanist sans-serif, will make for a balance between readability and modern design
- Use location-based icons (pins, building, etc.) to represent listings on the map.
- Map-centric layout with clear filters and property details.
- Smooth transitions when loading listings or moving on the map.
- Top toolbar for quick actions and settings.
- Navigation bar appears on the left side on phone screen size.
- Navigation bar appears on the bottom on tablet screen size.
